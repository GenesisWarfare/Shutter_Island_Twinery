This project is an interactive narrative created with Twine, inspired by the psychological thriller *Shutter Island*.  
You play as U.S. Marshal Teddy Daniels, investigating the disappearance of a patient during a violent storm on a remote island.  
Your choices shape the investigation and reveal hidden truths about the island and about yourself.

## How to Play

- Play in this link [Itch.io Link](https://genesiswarfare.itch.io/shutter-island) or download the HTML file from this repository.  
- Open it in your web browser (Chrome, Firefox, Edge, etc.).  
- Click the interactive links to make choices.  
- Explore different paths to discover multiple story outcomes.

## Project Structure

- **Shutter_Island.html**, the full interactive Twine story (playable in the browser)

## Tool Used

 **Twine** used to write and link the interactive passages  

## Features

- Branching storyline  
- Multiple routes and endings  
- Psychological and cinematic atmosphere  
- Narrative written in French
